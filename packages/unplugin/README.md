# @better-css-modules/unplugin

Internal shared package that provides the unplugin factory for better-css-modules bundler plugins.

This package is **private** and not published to npm. It is used internally by `@better-css-modules/vite`, `@better-css-modules/webpack`, `@better-css-modules/rollup`, `@better-css-modules/rspack`, and `@better-css-modules/esbuild`.
